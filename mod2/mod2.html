Learning Objectives

By the end of this module, students will be able to:

  1. Identify common web design tools and software.
    aligns to course learning outcome (CLO) (2)
  2. Demonstrate using web design tools to construct web pages.
    aligns to course learning outcome (CLO) (2)

Overivew

This module will explore the essential tools and software web designers use to create engaging and interactive websites. From graphic design programs to code editors, we will discover how these tools empower designers to bring their creative visions to life on the web. Whether you're a beginner or an experienced designer looking to expand your toolkit, this module will provide you with the knowledge and skills needed to excel in web design. Let's dive in!

Quiz

The assessment consists of 5 questions with a passing score set at 75%.

Assignment

Students will create a one-page website using Figma. The website will serve as a showcase of their skills, projects, and personality. The goal is to create a visually appealing and user-friendly design that highlights their work and experience.

Discussion

Imagine you are tasked with redesigning a popular social media platform's interface. How would you use Figma to collaborate with team members and create interactive prototypes? Discuss the key features of Figma that would be most beneficial for this project and how you would apply design principles to ensure a user-centric design.
