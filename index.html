Welcome to Building User-Centric Websites

Building User-Centric Websites is an engaging course that explores the principles and practices of web design, focusing on creating websites that prioritize the user experience. Whether you're a beginner or an experienced web developer, this course will equip you with the skills and knowledge to design websites that are visually appealing but also intuitive and user-friendly. Throughout this course, you'll learn about the key elements of web design, including layout, color, typography, and user experience. You'll also be able to practice using popular web design tools and software, such as Figma and Sublime Text, to bring your creative ideas to life.

By the end of this course, you'll have the knowledge and skills to design professional-looking websites that look great and provide an excellent user experience. So, get ready to embark on an exciting journey into the world of web design, and let's start creating amazing web layouts together!

Course Description

Building User-Centric Websites is a mini-course designed to introduce learners to the principles and practices of web design, with a focus on creating websites that are accessible and user-friendly across different devices. The course covers web design principles, typography, color theory, web accessibility, and user-centric design.
Course Prerequisite(s): Familiarity with web design concepts is helpful but not required.

Course Learning Outcomes

By the end of this course, students will be able to:

  1. Explain the fundamental principles of web design, web design layout, color schemes, and typography.
  2. Apply web design tools and software to construct functional and visually appealing web pages.
  3. Identify web accessibility and implement best practices to create user-friendly, inclusive websites.
  4. Evaluate and analyze existing websites for accessibility issues and propose effective improvement solutions.
  5. Create a user-centric website, ensuring the incorporation of accessibility features for an inclusive user experience.
